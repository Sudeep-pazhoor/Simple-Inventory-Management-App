## Inventory Management System

A simple app to manage your products — add, view, edit, and delete them with ease.

## What You'll Need

- Node.js and npm installed
- A code editor

## Setup

# Set up the backend
cd server
npm install

# Set up the frontend 
cd client
npm install

## Running the App

//Backend
cd server
npm start

//Frontend
cd client
npm run dev


Once both are running, open your browser and go to **http://localhost:5173**.

## What Can You Do With It?

- Sign up / Log in— your data stays private to you
- View all your products on a clean dashboard
- Add, edit, or delete products.

## Project Structure

Inventory/
├── client/    # Everything you see in the browser
└── server/    # The backend that stores and serves your data
